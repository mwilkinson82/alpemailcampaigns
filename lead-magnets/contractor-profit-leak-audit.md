## Audit Structure

Five sections.

Five questions per section.

Use a 1-5 scoring scale.

```text
1 = Never true
2 = Rarely true
3 = Sometimes true
4 = Usually true
5 = Always true
```

Important:

For this audit, the lowest-scoring category is the biggest leak. Low score means the system is missing or weak.
