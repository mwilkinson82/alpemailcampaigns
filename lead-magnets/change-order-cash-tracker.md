## Role In The Machine

It exists to capture the fastest money pain and route the contractor into the Contractor Profit Leak Audit and Contractor Circle.

## Tracker Columns

| Column | Purpose |
| --- | --- |
| Job name | Which project the change belongs to |
| Description | What changed |
| Requested by | Owner, GC, architect, client, PM |
| Date discovered | When the change appeared |
| Submitted amount | Dollar value |
| Status | Draft, submitted, approved, billed, collected |
| Owner | Who is responsible |
| Aging | How long it has been sitting |
| Cash exposure | Money at risk |
