# Supporting Lead Magnet: Estimating Profit Checklist

## Role In The Machine

This is a supporting lead magnet.

It is a lite public diagnostic version, not the full Estimating Profit system.

It captures contractors who want more or better work, then teaches that estimating is the first profit system.

It should route back to the Contractor Profit Leak Audit and Contractor Circle.

The full Estimating Profit system stays inside Contractor Circle Hub.

## Public Promise

```text
Do not estimate just to win. Estimate to make money.
```

## Checklist

| Review item | Complete |
| --- | ---: |
| Scope is clear enough for production to build from it. |  |
| Exclusions are written clearly. |  |
| Assumptions are written clearly. |  |
| Markup and target gross profit are reviewed. |  |
| Labor risk is reviewed. |  |
| Material volatility is considered. |  |
| Subcontractor quotes are complete and current. |  |
| Change-order triggers are identified before the job starts. |  |
| Schedule risk is reflected in the number or terms. |  |
| Payment terms and billing milestones are reviewed. |  |
| Pre-send review is complete. |  |

## Three Pre-Send Questions

```text
Where can this job lose money?
What has to be clarified before we send the number?
What must become a change order if it happens later?
```
