# Premium Bounded Access

## Rule

No open-ended one-on-one.

Ever.

Premium access should be expensive, fixed-scope, and supportive of the ecosystem.

It should:

- Fund R&D.
- Create case studies.
- Validate the system.
- Produce implementation lessons.
- Improve the tools.
- Support the brand.

It should not consume the founder, hijack the roadmap, or become the core business.

## Offers

### 90-Day Contractor Operating Intensive

Price:

```text
$25k-$50k
```

Scope:

- Fixed 90-day engagement.
- Defined starting diagnosis.
- One implementation roadmap.
- Limited strategic review.
- Tool adoption plan.
- No unlimited access.

Best for:

- Serious growth-stage contractors who need bounded help installing the system.

### Annual Contractor Boardroom

Price:

```text
$75k-$150k/year
```

Scope:

- Quarterly strategy days.
- Limited async review.
- Contractor Circle / app subscription required.
- Fixed agenda.
- No ad hoc consulting sprawl.

Best for:

- Larger outliers who qualify for premium access.

### Private Implementation License

Price:

```text
$100k+/year
```

Scope:

- For companies with internal operators.
- Access to implementation pathways.
- Advisory to the operating team, not babysitting the owner.
- Limited strategic checkpoints.

Best for:

- Larger companies that need to deploy the doctrine internally.

## Qualification

Premium buyers must:

- Be inside Contractor Circle or maintain active tool access.
- Have enough internal capacity to implement.
- Agree to a fixed scope.
- Produce useful learning for the ecosystem.

## Anti-Patterns

Do not sell:

- Unlimited founder access.
- Random one-off consulting.
- Custom work that does not improve the product.
- Advisory that bypasses Contractor Circle and tools.
- Founder-dependent rescue work.
