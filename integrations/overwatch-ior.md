# OverWatch / IOR Integration

OverWatch / IOR is the project financial control product.

It does not own public lead magnets or drip campaigns.

## OverWatch / IOR Owns

- Project financial control workflows.
- IOR tools.
- OverWatch dashboards.
- Risk, dollars, project profit, and PM financial visibility.
- Project Profit Snapshot workflows.

## Campaign Events To Send

- `overwatch_signup_started`
- `overwatch_customer_created`
- `ior_tool_used`
- `project_profit_snapshot_created`

## Campaign Relationship

Public campaign assets can educate leads on project profit leaks, PM financial visibility, change-order exposure, and risk management.

Full IOR and OverWatch tools stay gated inside OverWatch / IOR or Contractor Circle Hub.
