# Contractor Circle Growth Machine

## Master Flywheel

```text
Public content shows them where money is leaking.
Lead magnets diagnose the leak.
Contractor Circle gives them the tools and implementation environment to fix the leak.
Calls keep members moving.
Member wins become new content.
```

## Simple Mental Model

```text
CONTENT creates attention.
LEAD MAGNET creates diagnosis.
EMAIL creates education and urgency.
CONTRACTOR CIRCLE creates implementation.
TOOLS create retention.
CALLS create wins.
WINS create proof.
PROOF creates more content.
CONTENT creates more attention.
```
