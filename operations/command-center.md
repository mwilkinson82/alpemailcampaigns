# Contractor Circle Command Center

## Operating Command

```text
Find the leak. Fix the leak. Install the system.
```

Public content says:

```text
Here is the leak.
```

Lead magnets say:

```text
Here is your leak.
```

Email says:

```text
Here is why the leak keeps happening.
```

Contractor Circle says:

```text
Here is the tool, rhythm, and accountability to fix it.
```

Calls say:

```text
Here is what you are implementing this week.
```

Dashboard says:

```text
Here is whether the machine is working.
```

Access rule:

```text
Public gets diagnosis and lite tools.

Contractor Circle Hub gets implementation and full tools.
```

See:

- `operations/access-model.md`
- `campaign-ops/README.md`

### 6. Email Automations

Purpose:

- Deliver results, educate, and convert to Contractor Circle.
- Use Resend as the delivery and automation layer.
- Keep campaign source-of-truth in a separate campaign repository, not inside OverWatch/IOR, AOS, or Contractor Circle Hub.

Source files:

- `campaigns/email-automation-setup.md`
- `campaigns/drip-campaigns.md`
- `campaign-ops/repository-architecture.md`
- `campaign-ops/resend-infrastructure.md`
- `campaign-ops/automations/resend-map.md`
- `campaign-ops/emails/master-profit-leak-sequence/README.md`

Minimum viable automation:

```text
Audit submitted -> leak tag applied -> Day 0 result email -> master sequence -> Contractor Circle CTA.
```

Infrastructure rule:

```text
Git tracks the plan and copy.
Resend sends and automates.
The dashboard tracks behavior and conversion.
The product apps fulfill access.
```

Access note:

```text
The public audit can capture leads. The Hub audit routes paid members into tools, assignments, calls, and implementation tracks.
```

Do not build all low-ticket products right now.

Do not split attention across the Estimating Profit Pack, Change Order Cash Kit, Project Profit Control Sprint, Company Control Fast Start Kit, and Contractor Money Systems Manual.

Right now, the practical order is:

```text
1. Free Profit Leak Audit
2. Free Change Order Cash Tracker lite
3. Contractor Circle
4. Low-ticket kits later
```
