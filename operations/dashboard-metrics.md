# Operating Dashboard

## Purpose

Track whether the growth machine is producing demand, converting buyers, activating members, and creating proof.

The key metric is not just members.

The key metric is:

```text
Active members using tools and reporting wins.
```

That is what creates retention, referrals, testimonials, and social proof.

## Command Center Tabs

Build the operating center with these sections:

1. Master Dashboard.
2. Lead Magnet Buildout.
3. Content Calendar.
4. Call Calendar.
5. Member Board.
6. Email Automations.
7. Asset Library.
8. Weekly Review.

The tool can be Notion, Airtable, ClickUp, Google Sheets, Circle, or another system. The structure matters more than the software.

## Resend Tracking Layer

Resend should be the source for public email events:

- Sent.
- Delivered.
- Opened, if tracking is enabled.
- Clicked, if tracking is enabled.
- Bounced.
- Complained.
- Suppressed.
- Unsubscribed.

Use contact properties and segments for leak routing. Use webhook events or exports to feed the operating dashboard.
